# Reflection State
<!-- howard@pincham.net 3/2026 -->


This file stores the current state of a reflective coaching cycle.

This file acts as a portable checkpoint for the reflective coaching process.
It allows the user to pause and resume the method across AI sessions.

At the end of a session or week, the AI coach may generate an updated
version of this file.

Save it somewhere safe.  
To continue the process in a new chat session, paste the entire file
into the conversation and ask the AI coach to resume.
