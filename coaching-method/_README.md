# Quiet Reflective Coaching Technique

This repository contains a conversational coaching methodology designed to support reflection, imagination, and discernment through structured dialogue with AI.

The method emphasizes:

- reflection over advice
- curiosity over urgency
- imagination over optimization

It combines a **Coach + Mirror** conversational posture with a structured reflective process that typically unfolds across a four-week exploration cycle.

## How to Use

1. Read the overview documents.
2. Copy the prompt from `META_PROMPT_V1.md`.
3. Paste it into a ChatGPT conversation.
4. Begin a reflective dialogue.

## Purpose

This technique is intended to help individuals explore questions involving:

- career transitions
- leadership reflection
- rediscovering imagination
- complex professional challenges

It is not intended to replace therapy, counseling, or professional advisory services.
